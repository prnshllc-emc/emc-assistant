📡 API REST

Documentação interativa disponível em /docs (Swagger) e /redoc (ReDoc).

Auth

POST /api/v1/auth/login  →  {"email":"...","password":"..."} → JWT
GET  /api/v1/auth/me     →  User info (Bearer token)

Items (CRUD)

GET    /api/v1/items          →  List (?urgency=, ?status=)
GET    /api/v1/items/{id}     →  Detail
POST   /api/v1/items          →  Create
PUT    /api/v1/items/{id}     →  Update
DELETE  /api/v1/items/{id}     →  Delete
POST   /api/v1/items/{id}/resolve → Toggle

Containers

GET   /api/v1/containers      →  List
POST  /api/v1/containers      →  Create
PUT   /api/v1/containers/{id} →  Update
