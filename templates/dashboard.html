📡 API REST

API documentation available at /docs (Swagger UI) and /redoc (ReDoc).

Auth

POST /api/v1/auth/login  →  {"email":"...","password":"..."} → JWT token
GET  /api/v1/auth/me     →  User info (requires Bearer token)

Items (CRUD)

GET    /api/v1/items          →  List (filtros: ?urgency=, ?status=, ?category=)
GET    /api/v1/items/{id}     →  Detail
POST   /api/v1/items          →  Create
PUT    /api/v1/items/{id}     →  Update
DELETE  /api/v1/items/{id}     →  Delete
POST   /api/v1/items/{id}/resolve → Toggle resolved
GET    /api/v1/items/stats    →  Dashboard stats

Containers

GET   /api/v1/containers      →  List all
POST  /api/v1/containers      →  Create
PUT   /api/v1/containers/{id} →  Update
